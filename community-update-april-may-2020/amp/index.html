Community Update: April & May 2020

by MESG

2020-06-03

Research & Development

  • Simplified and reorganized the Engine architecture to be CLI-based.
  • Improved Cosmos events in MESG modules
  • Moved all CLI, Framework documentation to Liteflow
  • Moved all js libraries and services to Liteflow
  • Improved MESG CLI for local development
  • Released new runner gRPC API
  • Removed gRPC API from the mesg/api library
  • Released MESG Aragon app for sending notifications
  • Improved CLI commands
  • Simplified the container package
  • Added logs to the Orchestrator to expose in gRPC stream
  • Tested integrations with Firebase, Google Cloud and App Engine
  • Updated Stripe, Zapier, Slack, Twilio, Twitter, Ethereum, webhook, encryption, and email services
  • Worked on improving Docker build time

Marketing & Communications

  • Updated & simplified MESG.com
  • Teased and announced Liteflow, a new framework and hosting solution built atop MESG
  • Updated community on MESG Token changes
  • Released tutorial on migrating MESG apps to Liteflow
  • Created demo video to set up Telegram notifications on Aragon
  • Worked on optimizing funnels, conversion, and analytics

Partners & Administration

  • Delisted the MESG Token from exchanges
  • Released Liteflow built on MESG technology for easy building deployment and monitoring.
