Introducing MESG Orchestrator

MESG Orchestrator is an event-driven task-orchestration system that manages the connections between the events and tasks of services by following the steps described in a simple process file.

It lets you build and run complex applications by describing their business logic, then the MESG Engine executes them for you.

In short, it is a decentralized way to build software faster. Create features or entire applications featuring complex or hard-to-reach technologies like blockchains, P2P networks or AI in software without needing to manage or even learn those technologies first.

Start building now

MESG Orchestrator is ready to use now. Check out the docs to learn more and start building.
